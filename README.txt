FRUIT AND VEGETABLE FRRESHNESS CLASSIFICATION

This project is designed to classify fruits and vegetables based on their type, freshness stage and estimate the 
number of days remaining until they become rotten. The model used is a VGG16 trained on a custom dataset of fruits. 
The backend is built with Flask and the frontend is built with React Native

INSTALLATION
Prerequisites:
1. Python 3.7 or higher
2. Node.js and npm
Expo CLI

Backend (Flask)
1. Install dependencies with ' pip install -r requirements.txt
2. Run the Flask app with ' python app.py '

Frontend (React Native)
1. Navigate to the frontend directory
2. Install dependencies with ' npm install '
3. Run app with ' npx expo start '
